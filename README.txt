# Discord Bot - Sistema de Administração e Diversão

## Descrição
Este é um bot para Discord desenvolvido inteiramente em JavaScript. Ele é projetado para auxiliar na administração do servidor, engajar a comunidade e oferecer recursos interativos, como um sistema de níveis para os usuários. Além disso, o bot integra-se com uma API que realiza buscas em um banco de dados de jogos, permitindo que os usuários encontrem informações detalhadas sobre qualquer jogo diretamente no servidor.

---

## Funcionalidades
- **Administração do Servidor**:  
  Comandos para gerenciar canais, papéis, banimentos e outras configurações do servidor.

- **Sistema de Níveis**:  
  Usuários ganham experiência ao interagir no servidor. Suba de nível e acompanhe sua progressão.

- **Pesquisa de Jogos**:  
  Integração com uma API externa que permite buscar informações sobre qualquer jogo. Resultados detalhados diretamente no chat.

- **Banco de Dados SQLite**:  
  Utilizado para armazenar dados de usuários, níveis, e configurações personalizadas do servidor.

---

## Tecnologias Utilizadas
- **[Node.js](https://nodejs.org)**: Ambiente de execução para JavaScript.
- **[Discord.js](https://discord.js.org)**: Biblioteca para criar bots no Discord.
- **[SupaBase](https://supabase.com)**: SGBD Online.
- **Axios**: Para realizar chamadas HTTP e integrar com a API de busca de jogos.
- **NestJs**: Backend/API feito em Nest integrado outras aplicações que eu faço.
